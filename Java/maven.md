## mvn 命令
    -v      查看maven版本
    compile 编译
    test    测试
    package 打包
    clean   删除target
    install 安装jar包到本地

### mvn命令创建项目
    1. archetype:generate   按照提示进行选择
    2. archetype:generate 
                -DgroupId=组织名，公司网址的反写+项目名
                -DartifactId=项目名-模块名
                -Dversion=版本号
                -Dpackage=代码所在的包名
